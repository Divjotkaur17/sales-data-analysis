# sales-data-analysis
Sales dashboard using Excel


Sales Data Analysis Dashboard;
Project Overview:-
This project analyzes retail sales data to identify trends in sales, profit, and product performance.

Tools Used :-
Microsoft Excel
Pivot Tables
Data Visualization

Key Insights:-
Technology category generates the highest sales
West region contributes the most profit
A small number of products drive most of the revenue
Several products are loss-making
High sales does not always mean high profit

Business Recommendations:-
Reduce discounts on loss-making products
Focus on high-profit products
Improve performance in low-profit regions
Optimize pricing strategy
